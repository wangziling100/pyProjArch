# pyProjArch
A build-tool used to build an empty common python project
## Installation
bash install.sh  
source $HOME/.bashrc
## Usage
pyBuild [-w your workspace] [project name] 
## Directory structure
project_name  
|-- bin
|    |
|    -- your_exec_file
|-- docs
|    |
|    -- abc.rst
|    |
|    -- conf.py
|-- init.py
|-- main.py
|-- README.md
|-- requirements.txt
|-- setup.py
|-- tests
|    |
|    -- init.py
|    |
|    -- test_main.py
|-- your_project_directory
